Docker Handler Module

The docker_handler module provides a wrapper around the Docker SDK with enhanced error handling and type safety.

Wrapper around Docker SDK client with enhanced error handling.

Provides connection management and error translation.

Examples

Basic connection:

>>> from docker_handler import DockerClientWrapper
>>> client = DockerClientWrapper()
>>> client.ping()
True

Using context manager:

with DockerClientWrapper() as client:
    containers = client.list_containers()
    for container in containers:
        print(f"{container.name}: {container.status}")
